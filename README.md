# native-tcp

## Start Project
### Start in Android
```bash
npm run android
```

### Start in iOS
```bash
npm run ios
```



## Build Project
### Build for iOS
```bash
eas build --platform ios --auto-submit
```


## Helpers
### Reverse ADB
```bash
adb reverse tcp:3000 tcp:80  
```

### Run YPC99 Simulation
```bash
npm run simulation
```
